# 示例代码接口索引

本技能提供四种语言（Java / Python / Go / PHP）的汇聚支付API代码示例。

## 接口列表

| 序号 | 接口 | API路径 | 功能 | 核心必填参数 |
|------|------|---------|------|-------------|
| 1 | 统一支付下单 | `/tradeRt/uniPay` | 发起支付 | p1/p2/p3/p5/q1/q9/qa/hmac |
| 2 | 订单查询 | `/tradeRt/queryOrder` | 查询支付结果 | p1/p2/hmac |
| 3 | 退款申请 | `/tradeRt/refund` | 发起退款 | p1/p2/p3/p4/p5/hmac |
| 4 | 退款查询 | `/tradeRt/queryRefund` | 查询退款状态 | p1/p2(hmac用refundOrderNo)/hmac |
| 5 | 退款信息查询 | `/tradeRt/queryRefundInfo` | 查询全部退款记录 | p1/p2/hmac |
| 6 | 关闭订单 | `/tradeRt/closeOrder` | 关闭未支付订单 | p1/p2/p3/hmac |
| 7 | 资金查询 | `/tradeRt/queryFundsControlOrder` | 发货管理/资金查询 | p1/p2/hmac |

## 语言特性对照

| 特性 | Java | Python | Go | PHP |
|------|------|--------|-----|-----|
| HTTP库 | HttpURLConnection / OkHttp | urllib (标准库) | net/http | cURL / Guzzle |
| JSON库 | fastjson / Jackson | json (标准库) | encoding/json | json_encode/decode |
| MD5库 | Apache Commons Codec | hashlib (标准库) | crypto/md5 | md5() |
| RSA库 | java.security / BouncyCastle | cryptography / rsa | crypto/rsa | openssl_* |
| JDK/运行时 | Java 8+ | Python 3.7+ | Go 1.16+ | PHP 7.4+ |
| 代码风格参考 | SignBiz.java (gw_pay项目) | uni_pay_client.py (现有) | 标准Go惯用法 | PSR-4/PSR-12 |

## 签名方式覆盖

每个语言的SDK工具类均包含：

| 文件 | 签名方式 | 说明 |
|------|---------|------|
| `JoinPaySignature` / `joinpay_signature.py` | **MD5** | HMAC-MD5签名，兼容现有CLI脚本 |
| `JoinPayRsaSignature` / `joinpay_rsa_signature.py` | **RSA** | SHA256WithRSA签名，兼容SignBiz.java |
| `JoinPayClient` / `joinpay_client.py` | **双模式** | 封装HTTP调用，内部自动选择签名方式 |

## 使用建议

1. **先读SDK工具类** — 理解签名和HTTP封装的实现
2. **再看接口示例** — 了解具体接口的参数组装方式
3. **参考项目实际代码** — Java版本参考 `gw_pay` 项目中的 `SignBiz.java` 和 `UniPayApiController.java`
4. **复制适配** — 示例代码仅供学习参考，需根据项目实际框架(Spring Boot/Django/ Gin/Laravel等)进行调整
